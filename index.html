Provincias Argentina API
Aqui encontraras todas las provincias de argentina con sus respectivos datos. Ciudad capital, Coordenadas, Población, Año de fundación, Fundador y Rango.

Documentación

Introducción

Esta documentación lo ayudará a familiarizarse con los recursos de la API de Provincias Argentina y le mostrará cómo hacer diferentes consultas.

REST

http://localhost:33528/api/v1/

La URL base contiene información sobre todos los recursos de API disponible. Todas las solicitides son GET. Todas las respuestas devolverán datos en json.

Sample Request: 

http://localhost:33528/api/v1/
{
    "provinces": "http://localhost:33528/api/v1/provinces/"
}

Actualmente hay un recurso disponible.

  • Provincias: se usa para obtener todas las provincias.

GraphQL

http://localhost:33528/graphql/

Sample Query: 

query {
    provinces {
        _id
        province
        foundation
        founder
    }
    province(id: 5) {
        _id
        capital
        province
        founder
    }
}

Acerca De

¿Que es esto?

Provincias Argentina API es una API RESTful y GraphQL, basada en los datos del INDEC. Tendrás acceso a los datos de la ciudad capital, fundación, fundador, coordenadas, población y rango.

¿Quién eres tú?

Soy Franco Andrés Sánchez, un Desarrollador Web que le gusta crear cosas y saber como funcionan.

¿Por qué construiste esto?

Porque estaba interesado en hacer una API con datos de argentina.

¿Derechos de autor?

Esta API es de código abierto y utiliza una licencia MIT. Puedes contribuir al proyecto en GitHub!

¿Cosas técnicas?

Estoy usando Node.js para servir la API, MongoDB como motor de base de datos. Todos los datos estan formateados en json. La API esta alojada en Microsoft Azure y este sitio en Netlify.
